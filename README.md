# Asali Swad Native Android App

This is a premium, native Android application built with **Kotlin** and **Jetpack Compose**.

## 🚀 Getting Started
1. Open this folder (`asali-swad-android`) in **Android Studio**.
2. Sync the project with Gradle files.
3. Configure your Supabase API keys in `SupabaseService.kt` (or use a `.env` / `BuildConfig` approach).
4. Run the app on an emulator or physical device.

## 🎨 Features
- **Modern UI**: Emerald brand colors with high-radius (32dp+) premium cards.
- **Compose Navigation**: Fluid transitions between screens.
- **Supabase Ready**: Pre-defined models and API interfaces for your backend.
- **Coil Integration**: Smooth image loading for product galleries.

## 🛠️ Project Structure
- `app/src/main/java/com/asaliswad/app/ui/theme`: Design system and brand tokens.
- `app/src/main/java/com/asaliswad/app/ui/screens`: Individual app screens (Home, Cart, Profile).
- `app/src/main/java/com/asaliswad/app/data`: Data models and network services.
